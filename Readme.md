# Project Generator (Personal Automation Tool)

Bash-based script used to scaffold frontend and backend projects 
with predefined structure and configuration.

This tool automates repetitive setup tasks such as:

- Installing required dependencies
- Configuring package.json scripts
- Writing base config files
- Overwriting template files
- Standardizing project structure

---

## Setup

```bash
chmod +x project-gen.sh
alias project-gen="~/folder_save/create_project.sh"
source ~/.bashrc
